# Release checklist

- [ ] Run `windows\tests\run-tests.ps1`.
- [ ] Confirm the archive contains no `windows/backups`, logs, screenshots, or `.lnk` files.
- [ ] Search text files for real names, drive-letter paths, tokens, email addresses, and private asset filenames.
- [ ] Confirm all bundled fonts retain their OFL files.
- [ ] Confirm LICENSE, NOTICE.md, PRIVACY.md, and ASSET_LICENSES.md are present.
- [ ] Test install, normal launch, compatible launch, and restore on a clean Windows account.
- [ ] Review the final archive contents before uploading.
